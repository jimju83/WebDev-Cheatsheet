HTML Tags


HTML Elements

Semantic Elements

In coding, semantics, refers to the meaning of code. Semantic elements in HTML describe the meaning of a tag to both the developer and the browser.

Name Tag Description
Header <header> The <header> element is a container for introductory content for the page or section.
Nav <nav> The <nav> element contains a set of navigational links.
Main <main> The <main> element contains the main content of a document.
Section <section> The <section> element is used to group related content together. It can be used to explicitly define a new section in the document outline
Article <article> An <article> defines an independent, or self-contained content.
Aside <aside> An <aside> element defines content that is indirectly related to the content that it placed next to (like a sidebar).
Figure <figure> The <figure> element represents self-contained content, like photos, diagrams, illustrations, code listings, etc. The <figcaption> tag defines a caption for the <figure> element.
Footer <footer> The <footer> element defines a footer for a document or sectioning.

A <footer> typically contains:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • links to the top
  • related content

Non-Semantic Elements

Non-semantic elements are similar to semantic elements, but do not tell the developer anything about the content

Name Tag Description
div <div> The <div> tag is used to make a division of content in a document
span <span> The <span> tag defines an inline container used to mark up part of a document.

HTML Table Reference


Table Tags

Name Tag Description
Table <table> The wrapper element for all HTML tables.
Table Head <thead> The set of rows defining the column headers in a table.
Table Body <tbody> The set of rows containing actual table data.
Table Row <tr> The table row container.
Table Data <td> The table row container.
Table Foot <tfoot> The set of rows defining the footer in a table.

Table Attributes

Name Tag Description
Column Span colspan Defines how many columns a td element should span.
Row Span rowspan Defines how many rows a td element should span.
