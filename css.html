Pseudo-Class Tags

CSS contains psuedo-classes to style link elements as signifiers to offer clues about how to interact with objects. The syntax for the psuedo-class is a:state

Name State Description
Link :link Normal unclicked state
Hover :hover Defines the behavior when the cursor is placed over the object.
Active :active Defines the behavior when the object is clicked
Visited :visited Defines the behavior when a link has been visited.

Name State Description
Before ::before Creates a psuedo-element that is the first child of the of the selected element. It can be used to add content with the content tag.
After ::after Creates a psuedo-element that is the first child of the of the selected element. It can be used to add content with the content tag.
